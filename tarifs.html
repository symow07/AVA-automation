Tout ce que vous devez savoirكل ما تحتاج معرفتهEverything you need to know

AVA fonctionne-t-il avec n'importe quel logiciel comptable ?هل تعمل AVA مع أي برنامج محاسبي؟Does AVA work with any accounting software?

Oui. AVA s'intègre via API avec Odoo, Sage, Microsoft Dynamics et tout système exposant une API ou exportant en CSV/Excel. Le setup inclut une phase d'intégration dédiée à votre système.نعم. تتكامل AVA عبر API مع Odoo وSage وMicrosoft Dynamics وأي نظام يعرض API أو يصدر CSV/Excel.Yes. AVA integrates via API with Odoo, Sage, Microsoft Dynamics and any system exposing an API or exporting CSV/Excel. Setup includes a dedicated integration phase for your system.

Comment se passe le setup initial ?كيف يتم الإعداد الأولي؟How does the initial setup work?

Notre équipe réalise l'intégration complète : connexion à votre système, configuration des règles PCM, paramétrage des alertes et formation de votre équipe. Durée : 2 semaines, sans interruption de votre activité.يقوم فريقنا بالتكامل الكامل: الاتصال بنظامك وتكوين قواعد PCM وإعداد التنبيهات وتدريب فريقك. المدة: أسبوعان.Our team handles the full integration: connecting to your system, configuring PCM rules, setting up alerts and training your team. Duration: 2 weeks, without interruption to your activity.

Mes données sont-elles sécurisées ?هل بياناتي آمنة؟Is my data secure?

Chaque cabinet a son propre environnement isolé avec des credentials séparés. Les données sont chiffrées en transit et au repos. Aucune donnée client n'est partagée entre cabinets.كل مكتب لديه بيئة معزولة خاصة ببيانات اعتماد منفصلة. البيانات مشفرة أثناء النقل وفي حالة السكون.Each firm has its own isolated environment with separate credentials. Data is encrypted in transit and at rest. No client data is shared between firms.

Puis-je changer de pack en cours d'abonnement ?هل يمكنني تغيير الباقة خلال الاشتراك؟Can I change pack during my subscription?

Oui, vous pouvez upgrader à tout moment. La différence de frais est proratisée sur le trimestre en cours. Il n'y a pas d'engagement minimum de durée.نعم، يمكنك الترقية في أي وقت. يتم احتساب فرق الرسوم بالتناسب على الربع الحالي. لا يوجد حد أدنى للالتزام.Yes, you can upgrade at any time. The fee difference is prorated for the current quarter. There is no minimum commitment period.

Les frais de setup sont-ils récurrents ?هل رسوم الإعداد متكررة؟Are setup fees recurring?

Non. Les frais de setup sont payés une seule fois lors du démarrage du projet. Seul l'abonnement trimestriel est récurrent.لا. رسوم الإعداد تُدفع مرة واحدة فقط عند بدء المشروع. فقط الاشتراك الفصلي هو المتكرر.No. Setup fees are paid only once at project start. Only the quarterly subscription is recurring.

Que se passe-t-il si un document n'est pas reconnu ?ماذا يحدث إذا لم يتم التعرف على مستند؟What happens if a document isn't recognized?

Si la confiance d'extraction est inférieure à 85%, le document est envoyé dans une file de revue humaine via Slack et Google Sheets — avec les données extraites pré-remplies pour correction rapide.إذا كانت ثقة الاستخراج أقل من 85%، يُرسل المستند إلى قائمة مراجعة بشرية عبر Slack وGoogle Sheets مع البيانات المستخرجة مسبقاً للتصحيح السريع.If extraction confidence is below 85%, the document is sent to a human review queue via Slack and Google Sheets — with pre-filled extracted data for quick correction.
